# Webpack4 从零开始搭建 Vue 环境

+ 针对单独的环境配置，请在分支里切换到 development 或 production 分支，单独查看。
+ production 分支的代码是在 development 分支基础上进行删减与新增改写的。
+ `npm run build` 打包代码后，可以执行 `npm run start` 启动服务查看效果。
+ 纯粹是做演示，所以页面非常难看，emmmm.... 不要老是去看页面。
+ 如果觉得有用点个 star 吧😊
